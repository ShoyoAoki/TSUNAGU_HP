# TSUNAGU コーポレートサイト 日中対訳用語集

対象: `TSUNAGU_HP` リポジトリの `/zh` 配下（コーポレートサイト中国語版）。
`messages/*.json` を持つ next-intl プロジェクト（Project-Bridg_1 / OwlMatch）とは別リポジトリだが、
ブランド用語は OwlMatch 本体（`/Users/aokishoyo/owlmatch/messages/zh.json`）の既存訳語と統一する。

新しい訳語を追加する場合は、必ずこの用語集に先に追記してから `src/app/zh/**` に反映すること。

## 0. 禁止語（法務・絶対厳守）

**「派遣」「劳务派遣」「人才派遣」等、労働者派遣を意味する語彙は日本語・中国語のいずれにおいても一切使用しない。**
株式会社TSUNAGUは労働者派遣事業の許可を取得しておらず、有料職業紹介事業（許可番号 46-ユ-300221）のみを行っているため。
実装後は必ず `grep -rn "派遣" src/` で 0件であることを確認する。

| 使ってよい表現 | 使ってはいけない表現 |
|---|---|
| 收费职业介绍 / 有偿职业介绍 | 劳务派遣 |
| 招聘 / 录用 / 职业介绍 | 人才派遣 |
| 远程试用（リモートトライアル） | 派遣 |

## 5. トーン方針

- 簡体字中国語はビジネス書面語・断定調。機械翻訳調（直訳・冗長・不自然な敬語）は禁止。
- 二人称は基本「您」（フォーム・法務文書）。マーケティングコピーでは「您」を省略し断定文で表現してもよい（ja側のコピー規則「断定文末」に準拠）。
- 「派遣」の禁止語チェックのほか、大学名（北京大学等）・加藤秀佳氏への言及はマーケティング文言に追加しない（既存メモリ方針に準拠）。
