# 画像フォルダ

このフォルダには、ウェブサイトで使用する画像ファイルを配置します。

## ロゴファイル

ヘッダーに表示されるロゴファイルを以下の名前で配置してください：

- **ファイル名**: `logo.svg`（推奨）または `logo.png`
- **推奨サイズ**: 幅 180px 程度（高さは自動調整されます）
- **形式**: SVG（ベクター形式、推奨）または PNG（透過背景推奨）

### SVGファイルの利点
- どのサイズでも高画質
- ファイルサイズが小さい
- 拡大縮小しても劣化しない

### 配置場所
```
public/
  └── images/
      └── logo.svg  ← ここに配置
```

ロゴファイルを配置後、サイトを再読み込みすると自動的に表示されます。
